## âœ… Step-by-Step Instructions

### Step 1: Select the Main Hero Container
1. Click on your **Main Hero Container** (the outermost container)
2. You should see a blue outline around it
3. Look for a **"+" button** or **"Add Container"** option

### Step 2: Add Container Inside
**Method A: Using the + Button**
1. When you hover over the main container, you'll see a **"+" button** appear
2. Click the **"+" button**
3. This adds a new container **inside** the main container

**Method B: Using Right-Click Menu**
1. Right-click on the main container
2. Select **"Add Container"** from the menu
3. A new container appears inside

**Method C: Using Widget Panel**
1. Make sure main container is selected
2. In the left widget panel, search for **"Container"**
3. Drag **"Container"** widget into the main container
4. Drop it inside

## âš ï¸ Common Issues & Solutions

### Issue 1: "+" Button Not Appearing
**Solution:**
- Make sure you're hovering over the container itself
- Try clicking the container first, then look for "+" button
- Check if container is selected (blue outline)

### Issue 2: Container Added Outside Instead of Inside
**Solution:**
- Delete the container you just added
- Make sure main container is selected/highlighted
- Try right-click method instead

### Issue 3: Can't Find Column Settings
**Solution:**
- Make sure you're in **Layout tab**
- Look for "Columns" or "Layout" dropdown
- Some versions: Look for "Structure" or "Grid" options
- Alternative: Add columns manually after creating container
