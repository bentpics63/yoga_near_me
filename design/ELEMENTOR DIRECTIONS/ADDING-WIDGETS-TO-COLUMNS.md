## üîç If Widgets Aren't Stacking

### Check 1: Are You Adding to the Column?
1. **Click inside the column** (not on spacer, but empty space)
2. Click **"Add Widget"** button
3. Widget should appear below existing widgets

### Check 2: Are Widgets Overlapping?
- If widgets overlap, check column settings
- Make sure column has proper spacing

### Check 3: Try Adding Below Spacer
1. **Click the Spacer widget**
2. Look for **"+" button** that appears below it
3. Click that **"+" button**
4. Add widget - it should appear below spacer

---

## ‚úÖ Method 1: Add Widgets Directly to Column

### Step 1: Click in Column
1. Click **inside the column** (empty space, not on spacer)
2. You should see **"Add Widget"** option or **"+" button**

### Step 2: Add Widget
1. Click **"Add Widget"** or **"+" button**
2. Search for widget (e.g., "GD Post Title")
3. Click widget name or drag it in
4. Widget appears below spacer

### Step 3: Add More Widgets
1. Click **"Add Widget"** again (below first widget)
2. Add next widget
3. They stack automatically

---

## ‚úÖ Method 2: Use + Button on Widgets

### If You See + Buttons:
1. **Hover over Spacer widget**
2. Look for **"+" button** that appears below it
3. Click **"+" button**
4. Add widget - it appears below spacer

### Then:
1. **Hover over the widget you just added**
2. **"+" button** appears below it
3. Click to add next widget
4. Continue adding widgets this way

---

## ‚úÖ Method 3: Add Container Inside Column (If Needed)

**Only if widgets truly won't stack:**

### Step 1: Add Container Widget
1. Click **"Add Widget"** in column
2. Search for **"Container"** widget
3. Add it to column

### Step 2: Add Widgets to Container
1. Click inside the container
2. Add widgets to container
3. They'll stack inside container

**Note:** This is usually unnecessary - widgets should stack in columns directly.

## üéØ What You Should See

**When adding widgets to a column:**
- Widget 1 appears
- Widget 2 appears **below** Widget 1
- Widget 3 appears **below** Widget 2
- They stack vertically automatically

## ‚ö†Ô∏è Troubleshooting

### Issue: Widgets Appear Side-by-Side
**Solution:**
- Make sure you're adding to the **column**, not creating new columns
- Check that column isn't set to horizontal layout

### Issue: Can't Click "Add Widget"
**Solution:**
- Click **empty space** in column (not on spacer)
- Or click **"+" button** that appears when hovering over widgets

### Issue: Widgets Replace Each Other
**Solution:**
- Make sure you're clicking **"Add Widget"** (not replacing)
- Use **"+" button** below existing widgets
