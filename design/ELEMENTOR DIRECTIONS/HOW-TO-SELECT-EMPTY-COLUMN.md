When a column is empty (no widgets), it's harder to click and select. Here's how to do it:

## âœ… Method 1: Click on Column Border/Edge

### Step 1: Find the Column Border
1. Look for the **border** or **edge** of the empty column
2. You should see a faint outline or border
3. **Click directly on that border/edge**

### Step 2: Column Should Select
- Blue outline should appear around the column
- Settings panel opens on left
- Now you can set width

---

## âœ… Method 2: Add Temporary Widget, Then Remove

### Step 1: Add Any Widget
1. Click **"Add Widget"** in the empty column
2. Add any widget (like "Text Editor" or "Spacer")
3. This makes the column easier to select

### Step 2: Set Column Width
1. Click the widget you just added
2. Then click the **column** (not the widget)
3. Set width: 66% or 34%

### Step 3: Remove Widget (Optional)
- You can delete the temporary widget after setting width
- Or keep it and add your real widgets

## âœ… Method 4: Hover and Look for Column Indicator

### Step 1: Hover Over Empty Column
1. **Hover your mouse** over the empty column area
2. You might see:
   - A faint outline
   - A "+" button
   - Column label/indicator

### Step 2: Click When Indicator Appears
1. When you see the outline or indicator, **click it**
2. Column selects
3. Set width

3. This makes column selectable

### Step 2: Select Column (Not Widget)
1. **Click outside the widget** but inside the column
2. Or click the **column border**
3. Column should select (blue outline)
