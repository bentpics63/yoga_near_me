## âš ï¸ Common Issues

### Issue 1: Can't Find Width Setting
**Solution:**
- Make sure you clicked the **COLUMN** (not the container)
- Try **Layout tab** first
- If not there, check **Advanced tab**
- Some versions: Width is in **"Flex"** or **"Grid"** settings

### Issue 2: Width Setting Shows Pixels, Not Percentages
**Solution:**
- Look for a dropdown next to the input (px/%/em)
- Change dropdown to **"%"**
- Then enter `70` or `30`

### Issue 3: Columns Don't Resize
**Solution:**
- Make sure container is set to **2 columns** first
- Try refreshing Elementor (save and reload)
- Check if there's a "Lock" or "Fixed" setting enabled
