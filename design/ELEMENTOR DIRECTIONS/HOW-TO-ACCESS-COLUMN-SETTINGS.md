# How to Access Column Settings in Elementor
## Alternative Methods When Gear Icon Isn't Visible

---

## ‚úÖ Method 1: Click Column ‚Üí Settings Panel Opens Automatically

In modern Elementor, clicking a column should **automatically open** the settings panel on the left.

### Steps:
1. **Click directly on the LEFT column**
2. **Look at the LEFT SIDE** of your screen - settings panel should open automatically
3. **No gear icon needed** - settings appear automatically
4. Look for **"Layout"** tab in the left panel
5. Find **"Width"** setting

---

## ‚úÖ Method 2: Right-Click Menu

1. **Right-click** on the column
2. Look for menu options like:
   - "Edit Column"
   - "Column Settings"
   - "Settings"
3. Click that option
4. Settings panel opens

---

## ‚úÖ Method 3: Use Elementor Navigator

1. Look for **"Navigator"** icon (usually bottom-left or hamburger menu)

---

## ‚úÖ Method 4: Double-Click Column

1. **Double-click** directly on the column
2. Settings panel should open
3. Look for Layout tab ‚Üí Width setting

## üéØ What to Look For

When you click a column, check:

### Left Side Panel:
- Should show column/widget settings
- Tabs at top: **Content**, **Style**, **Layout**, **Advanced**
- Click **"Layout"** tab
- Look for **"Width"** section

### Column Outline:
- When selected, column should have **blue outline**
- This confirms it's selected

---

## üìê Finding Width Setting

Once settings panel is open:

1. **Click "Layout" tab** (if not already open)
2. Look for:
   - **"Width"** dropdown
   - **"Column Width"** input
   - **"Flex"** or **"Grid"** settings
   - **"Size"** option

3. **Common locations:**
   - Layout ‚Üí Width ‚Üí Custom ‚Üí Enter `70%`
   - Layout ‚Üí Flex ‚Üí Width ‚Üí `70%`
   - Advanced ‚Üí Custom CSS ‚Üí `width: 70%`

---

## ‚ö†Ô∏è If Settings Panel Doesn't Open

### Try These:
1. **Click column** - wait a moment for panel to appear
2. **Click outside** then click column again
3. **Check if panel is collapsed** - look for arrow/chevron to expand
4. **Look for panel toggle** - might be hidden/minimized

### Panel Location:
- Usually on **LEFT side** of screen
- Sometimes **RIGHT side** (check both)
- Might be **minimized** - look for expand arrow

---

## üéØ Quick Test

1. **Click the LEFT column**
2. **Look LEFT side of screen** - do you see a settings panel?
3. If yes ‚Üí Click **"Layout"** tab ‚Üí Find **"Width"**
4. If no ‚Üí Try right-clicking column or double-clicking

## ‚úÖ Alternative: Use CSS Method

If you can't find width settings:

1. Click column
2. Go to **"Advanced"** tab
3. **Custom CSS** section
4. Add: `width: 70% !important;`

Do this for:
- Left column: `width: 70% !important;`
- Right column: `width: 30% !important;`
