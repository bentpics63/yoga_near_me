## Primary Content Types

### 1. Studio Listings
URL Pattern: /studios/{studio-slug}/
Schema: LocalBusiness, YogaStudio
Content: Studio name, address, phone, hours, yoga styles offered, amenities, photos, reviews
Example: https://yoganearme.info/studios/viveka-yoga-studio/

### 2. Location Archives
URL Pattern: /location/{country}/{state}/{city}/
Content: List of yoga studios in a geographic area with filtering options
Example: https://yoganearme.info/location/united-states/california/los-angeles/

### 3. Yoga Glossary
URL Pattern: /yoga-glossary/{term-slug}/
Schema: DefinedTerm, Article
Content: Sanskrit/yoga term definitions with pronunciation, etymology, practice context
Terms Covered: 70+ yoga concepts
Example: https://yoganearme.info/yoga-glossary/pranayama/

### 4. Yoga Style Guides
URL Pattern: /yoga-styles/{style-slug}/
Content: Comprehensive guides to yoga styles (Vinyasa, Hatha, Kundalini, etc.)
Includes: Origins, philosophy, class structure, benefits, who it's for
Example: https://yoganearme.info/yoga-styles/vinyasa/

### 5. FAQ Content
URL: https://yoganearme.info/30-essential-questions-answered/
Content: Common questions about starting yoga, finding studios, practice tips

### 6. Blog
URL: https://yoganearme.info/blog/
Content: Articles for practitioners and studio owners
