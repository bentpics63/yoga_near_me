# **Yoga Near Me \- Blog Design Style Guide**

## **Design Philosophy**

Create a minimalist, clean blog design that feels warm, welcoming, organic, and optimistic. The design should match the yoganearme.info homepage aesthetic while providing an excellent reading experience for athletic-minded individuals seeking spiritual depth outside traditional Western religious frameworks.

---

## **Brand Colors**

### **Core Palette**

\--primary-color: \#5f7470;     /\* Muted teal \- main brand color \*/  
\--text-primary: \#393939;      /\* Dark gray \- body text \*/  
\--accent-bg: \#E0E2DB;         /\* Cream \- backgrounds, soft sections \*/  
\--secondary-color: \#61948B;   /\* Teal \- supporting elements \*/

### **Highlight Colors (Style-Specific Use)**

\--highlight-coral: \#ff5733;   /\* Energetic styles (Vinyasa, Power) \*/  
\--highlight-purple: \#706677;  /\* Meditative styles (Yin, Restorative) \*/  
\--highlight-sage: \#B8BD5;     /\* Balanced styles (Hatha) \*/  
\--highlight-gold: \#FFD966;    /\* Warm styles (Kundalini, Hot Yoga) \*/

### **Usage Guidelines**

* **Background**: White primary, \#E0E2DB for alternating sections  
* **Headers**: Use \#5f7470 as default, BUT incorporate highlight colors based on yoga style being featured  
* **Body text**: \#393939 throughout  
* **Accents/buttons**: Use highlight colors for visual interest and to differentiate yoga styles  
* **Links**: \#61948B with hover state using relevant highlight color

---

## **Typography**

### **Fonts**

**Body Text**: Roboto (400, 500 weights)

* Clean, modern, highly readable  
* Size: 18px base for comfortable reading  
* Line height: 1.7 for long-form content

**Headers**: Choose a contrasting display font that feels:

* Confident and committed (not timid)  
* Organic and natural (not overly geometric)  
* Warm but sophisticated  
* **Suggested options**: Playfair Display, Lora, Merriweather, or Cormorant Garamond  
* Alternative: Use Roboto Black (900 weight) with dramatic letter spacing

### **Type Scale**

H1 (Article Title): 48-56px, bold/heavy weight  
H2 (Major sections): 36-40px, bold weight    
H3 (Subsections): 28-32px, semibold weight  
H4 (Minor headers): 22-24px, medium weight  
Body: 18px, regular weight  
Caption/Meta: 14-16px, regular weight

### **Typography Treatment**

* **Letter spacing on headers**: 0.02-0.05em (slight tracking for elegance)  
* **Text rendering**: Optimize for legibility (-webkit-font-smoothing: antialiased)  
* **Line length**: 60-75 characters per line for optimal readability  
* **Paragraph spacing**: 1.5-2em between paragraphs

---

## **Layout Structure**

### **Content Width**

**Optimize for reading comfort:**

* Desktop: 720-800px max width for body content  
* Wide sections (images, quotes): Up to 1200px  
* Sidebar (if used): 280-320px

### **Responsive Breakpoints**

* Mobile: \< 768px  
* Tablet: 768px \- 1024px  
* Desktop: \> 1024px

### **Section Spacing**

* Between major sections: 80-120px  
* Within sections: 40-60px  
* Paragraph spacing: 24-32px  
* Clean white space is essential \- don't overcrowd

### **Grid System**

* Use clean, minimal grid  
* Plenty of breathing room  
* Asymmetrical layouts okay for visual interest  
* Images can break grid occasionally for dynamic feel

---

## **Component Specifications**

### **Hero Section**

**Style**: Match yoganearme.info homepage aesthetic

* Full-width background image  
* Overlay: subtle gradient using primary color (\#5f7470) at 20-30% opacity  
* Title: Large, bold, clear hierarchy  
* Subtitle: Lighter weight, slightly smaller  
* Breadcrumbs: Small, upper left or centered above title  
* Height: 60-80vh on desktop, 50-60vh mobile

### **Header Colors by Yoga Style**

**Dynamic color system** \- use highlight colors to reflect style energy:

* **Hatha**: \#B8BD5 (sage green) \- balanced, foundational  
* **Vinyasa/Power**: \#ff5733 (coral) \- energetic, dynamic  
* **Yin/Restorative**: \#706677 (purple-gray) \- calm, meditative  
* **Hot/Bikram**: \#FFD966 (gold) \- warm, intense  
* **Ashtanga/Iyengar**: \#5f7470 (primary) \- traditional, structured  
* **Kundalini**: \#FFD966 (gold) \- spiritual, energetic

### **Content Blocks**

**Introduction Paragraph**:

* Larger font size (20-22px)  
* Optional: Subtle left border in accent color  
* Extra line height (1.8)

**Section Headers**:

* Use style-appropriate color from palette  
* Optional underline accent (3-4px, 60-80px width)  
* Margin: generous top spacing (60-80px), moderate bottom (24-32px)

**Highlight/Callout Boxes**:

* Background: \#E0E2DB (cream)  
* Border-left: 4-6px solid in style color  
* Padding: 32-40px  
* Border-radius: 0 or subtle (4-8px)  
* Box-shadow: Minimal or none for clean aesthetic

**Pull Quotes**:

* Larger text (24-28px)  
* Light or italic weight  
* Center-aligned or offset  
* Color: Style highlight color or primary  
* Optional quotation marks styled decoratively

**Lists**:

* Clean bullets or custom icons  
* Generous spacing between items (16-20px)  
* Optional: Light background for entire list block

**Images**:

* Full width or 2/3 width options  
* Caption: Small text (14-16px), italic, centered  
* Border-radius: 0 or subtle (4-8px)  
* Box-shadow: Subtle depth (0 2px 8px rgba(0,0,0,0.08))

**Videos**:

* 16:9 aspect ratio maintained  
* Subtle border or frame  
* Caption below

### **Buttons/CTAs**

* Primary button: Gradient or solid using style color  
* Border-radius: 50px (pill shape) or 8px (rounded rectangle)  
* Padding: 14-18px horizontal, 40-60px vertical  
* Font: Roboto, medium weight, 14-16px  
* Hover: Subtle scale (1.02) and shadow increase  
* Text transform: None or uppercase based on context

### **Table of Contents (Sidebar or Fixed)**

* Sticky positioning on scroll  
* Clean typography  
* Active section highlighted in style color  
* Smooth scroll to sections  
* Mobile: Collapsible or move to top

### **Author Bio**

* Background: \#E0E2DB  
* Avatar: Circular, 80-100px  
* Social links: Icons in primary color  
* Clean, uncluttered layout

### **Related Articles**

* Card-based design  
* Image thumbnail (16:9 ratio)  
* Title, excerpt, read time  
* Hover: Subtle lift and shadow  
* 3 column desktop, 1 column mobile

---

## **Visual Elements**

### **Icons**

* Style: Line icons or minimal solid icons  
* Color: Primary or style-specific highlight  
* Size: Consistent sizing within sections  
* Source: Font Awesome, Heroicons, or custom SVGs

### **Dividers**

* Minimal use  
* When used: 1px line in \#E0E2DB or style color at 20% opacity  
* Alternative: White space as primary divider

### **Shadows**

* Subtle and minimal  
* Cards/images: 0 2px 8px rgba(0,0,0,0.08)  
* Hover states: 0 4px 12px rgba(0,0,0,0.12)  
* Avoid heavy shadows \- keep organic and light

### **Borders**

* Generally minimal  
* When used: 1px solid \#E0E2DB  
* Accent borders: 3-6px solid in style color

---

## **Interaction Design**

### **Animations**

* Smooth and subtle  
* Scroll-triggered fade-ins for sections  
* Hover states: 0.3s ease transitions  
* No jarring or excessive motion  
* Respect prefers-reduced-motion

### **Reading Progress Bar**

* Fixed to top  
* Style color or gradient  
* 2-4px height  
* Smooth animation

### **Smooth Scrolling**

* Enable for anchor links  
* Ease timing for natural feel

### **Mobile Interactions**

* Touch-friendly targets (min 44x44px)  
* Swipe-friendly galleries  
* No hover-dependent interactions

---

## **Content Guidelines**

### **Voice & Tone**

* Warm but knowledgeable  
* Accessible without being condescending  
* Depth without density  
* Athletic meets spiritual  
* Inclusive of all backgrounds  
* Avoid Western religious terminology

### **Imagery Style**

* High quality, natural lighting  
* Diverse practitioners  
* Authentic yoga settings  
* Not overly posed or stock-photo feeling  
* Earth tones and natural environments  
* Show movement and stillness

### **Accessibility**

* WCAG 2.1 AA compliance minimum  
* Color contrast: 4.5:1 for body text, 3:1 for large text  
* Alt text for all images  
* Keyboard navigation support  
* Screen reader friendly markup  
* Focus indicators visible

---

## **Technical Requirements**

### **Performance**

* Lazy load images below fold  
* Optimize images (WebP with fallback)  
* Minimize CSS/JS  
* Target: \< 3s load time  
* Mobile-first approach

### **SEO**

* Semantic HTML5 structure  
* Proper heading hierarchy (only one H1)  
* Meta descriptions and titles  
* Structured data (Article schema)  
* Open Graph tags for social sharing

### **Browser Support**

* Modern browsers (last 2 versions)  
* Progressive enhancement  
* Graceful degradation for older browsers

---

## **Style-Specific Variations**

Each yoga style article should feel distinct while maintaining brand consistency:

1. **Use appropriate highlight color** from palette  
2. **Adjust header colors** to reflect style energy  
3. **Consider background variations**:  
   * Energetic styles: More white, brighter  
   * Meditative styles: More cream (\#E0E2DB), softer  
4. **Button colors**: Match style highlight  
5. **Hero overlay**: Can adjust opacity/color slightly per style

---

## **Do's and Don'ts**

### **DO:**

✓ Use generous white space ✓ Let content breathe ✓ Match homepage aesthetic ✓ Use highlight colors purposefully ✓ Optimize for reading comfort ✓ Keep navigation simple ✓ Use high-quality imagery ✓ Maintain consistent typography hierarchy

### **DON'T:**

✗ Overcrowd the layout ✗ Use all highlight colors on one page ✗ Make headers timid or uncommitted ✗ Sacrifice readability for style ✗ Use heavy shadows or gradients everywhere ✗ Create overly complex navigation ✗ Use stock photos that feel inauthentic ✗ Mix too many font families

---

## **Implementation Notes for Claude Code**

* Prioritize **reading experience** \- this is long-form content  
* Ensure **mobile optimization** is excellent, not an afterthought  
* Build **modular components** that can be mixed/matched  
* Create **style variations** easily swappable via CSS variables  
* Include **sample content** for each component type  
* Provide **Elementor Pro compatibility** in mind  
* Document **responsive breakpoints** clearly  
* Include **accessibility features** from the start

This design should feel like a natural extension of yoganearme.info while providing an exceptional reading experience for in-depth yoga content.
