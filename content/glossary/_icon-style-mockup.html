Core Style Rules

  • Smooth contoured line drawing ‚Äî anatomical shapes with body mass, NOT stick figures
  • Uniform stroke weight ‚Äî 1.5-2px standard
  • No facial features ‚Äî blank oval heads only
  • One color per icon ‚Äî from brand palette only
  • Curved, flowing lines ‚Äî shoulders, torso curves, natural limb shapes
  • Sanskrit symbols ‚Äî use when concept is abstract (prana, chakra, om)

When to Use Each Type

Pose Illustrations (Contoured Line Art Figures)

  • Glossary terms for specific poses (Chaturanga, Tree Pose, Warrior)
  • Style pages where physical practice is the focus (Hatha, Vinyasa)
  • How-to sections showing alignment
  • Must have anatomical form ‚Äî shoulders, torso curve, limb volume

Symbolic Icons (Sun/Moon, Breath Waves)

  • Concepts that aren't tied to a single pose
  • Pranayama and breathwork sections
  • Philosophy and tradition sections

Sanskrit Symbols (Devanagari Script)

  • Mantra definitions (Om, specific chants)
  • Highly abstract concepts (Prana, Chakra energy)
  • When no physical representation exists
  • Cultural/traditional emphasis needed

SVG Code Template

Standard SVG Structure for Contoured Figures

<svg viewBox="0 0 100 100" xmlns="http://www.w3.org/2000/svg">
  <!-- Head: smooth oval, no features -->
  <ellipse cx="50" cy="16" rx="10" ry="12"
    fill="none"
    stroke="#667085"
    stroke-width="1.5"/>

  <!-- Body: curved paths for anatomical form -->
  <!-- Use Q (quadratic) and C (cubic) bezier curves -->
  <!-- NOT straight lines for limbs -->
  <path d="M44 27 C38 30 28 36 24 44..."
    fill="none"
    stroke="#667085"
    stroke-width="1.5"
    stroke-linecap="round"
    stroke-linejoin="round"/>
</svg>

Key: Use bezier curves (C, Q commands) not straight lines (L). This creates the smooth, anatomical contours that match the Asana Image reference.
