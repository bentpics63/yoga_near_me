# BUILD SINGLE STUDIO PAGE IN ELEMENTOR

## IMPORTANT: The HTML demo was just a DESIGN REFERENCE
The HTML page I created shows what the final page should LOOK LIKE, but you need to BUILD it in Elementor first.

---

## STEP 1: PAGE STRUCTURE

### Header Section (if needed)
- Add Header widget or use your theme header
- Or skip if using theme header

### Hero Section
1. **Add Section** → Full Width
2. **Add Column** → Full Width
3. **Add Image Gallery widget** (2 Post Images widgets)
   - Main large image (left)
   - Two smaller images (right, stacked)
4. **Add Section** → Full Width
5. **Add Column** → Full Width
6. **Add widgets in this order:**
   - GD>Post Title widget
   - GD>Post Rating widget
   - Button widget: "Book a Class"
   - Button widget: "Save"
   - GD>Post Address widget
   - Share Buttons widget

### Main Content Section
1. **Add Section** → Boxed Width
2. **Add 2 Columns** (Main content left, Sidebar right)

#### Left Column (Main Content):
- **About Section:**
  - Heading widget: "About [Studio Name]"
  - Text Editor widget: About text
- **Yoga Styles Section:**
  - Heading widget: "Yoga Styles Offered"
  - Text Editor or HTML widget: Yoga style pills
- **Amenities Section:**
  - Heading widget: "Amenities & Features"
  - Text Editor or HTML widget: Amenities grid
- **Schedule Section:**
  - Heading widget: "Today's Schedule"
  - Text Editor or HTML widget: Schedule tabs and classes
- **Reviews Section:**
  - Heading widget: "Reviews"
  - GD>Post Rating widget (if available)
  - Text Editor widget: Review content

#### Right Column (Sidebar):
- **Contact Card:**
  - Heading widget: "Contact Information"
  - GD>Post Address widget
  - Text Editor widget: Phone, Email, Website
  - Share Buttons widget
- **Hours Card:**
  - Heading widget: "Hours of Operation"
  - Text Editor widget: Hours list
- **Map Card:**
  - Heading widget: "Location"
  - Map widget or Image widget (map placeholder)
  - Button widget: "Get Directions"
- **Claim Card:**
  - Heading widget: "Own This Studio?"
  - Text Editor widget: Claim text
  - Button widget: "Claim Listing"

### Nearby Studios Section
1. **Add Section** → Full Width
2. **Add Column** → Full Width
3. **Add Heading widget:** "Nearby Yoga Studios"
4. **Add Link widget:** "View All" (right side)
5. **Add 3 Columns** (for studio cards)
   - Each column: Studio card with image, name, location, rating

### Footer Section
- Use your theme footer
- Or add Footer widget section
