# Frontend Design Skill

This skill provides specialized assistance for frontend web development, particularly focused on modern web design, WordPress/Elementor integration, and creating engaging user experiences for yoga and wellness websites.

## Core Capabilities

### 1. HTML & CSS Development
- Semantic HTML5 structure
- Modern CSS techniques (Flexbox, Grid, Custom Properties)
- Responsive design patterns and media queries
- Cross-browser compatibility
- Accessibility best practices (WCAG compliance)

### 2. WordPress & Elementor
- Custom Elementor widget development
- WordPress theme customization
- Page builder optimization
- Custom CSS for WordPress sites
- Template hierarchy and structure

### 3. Design Systems
- Component-based design patterns
- Typography systems and hierarchies
- Color palette implementation
- Spacing and layout consistency
- Design token management

### 4. Responsive & Mobile-First Design
- Mobile-first CSS architecture
- Breakpoint strategies
- Touch-friendly interfaces
- Performance optimization for mobile
- Progressive enhancement

### 5. UI/UX Best Practices
- Visual hierarchy and information architecture
- User flow optimization
- Call-to-action placement and design
- Form design and usability
- Loading states and micro-interactions

### 6. Yoga & Wellness Website Specialization
- Calming, mindful design aesthetics
- Class schedule and booking interfaces
- Instructor profile layouts
- Blog and content presentation
- Gallery and portfolio displays
- Testimonial and review sections

## When to Use This Skill

Claude will automatically invoke this skill when you're:

- **Building web pages**: Creating new HTML pages or templates
- **Styling components**: Writing CSS, implementing designs
- **WordPress development**: Working with themes, Elementor, or custom templates
- **Responsive design**: Making layouts work across devices
- **UI improvements**: Enhancing user experience or visual design
- **Design systems**: Creating consistent, reusable components
- **Layout problems**: Solving spacing, alignment, or positioning issues
- **Accessibility**: Making content accessible to all users

## Design Principles for Yoga Websites

1. **Calm & Mindful**: Use whitespace generously, soft colors, and gentle animations
2. **Accessibility First**: Ensure content is readable and navigable for all users
3. **Performance**: Fast loading times, optimized images, minimal JavaScript
4. **Mobile-First**: Most yoga students browse on mobile devices
5. **Clear CTAs**: Easy booking, class schedules, and contact information
6. **Visual Storytelling**: Use imagery to convey the yoga experience
